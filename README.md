# twitter-cloning-firebase

twitter 클론코딩 (firebase)

- react 와 firebase를 이용해서 간단한 앱 작성

- eslint & prettier 로 코드 형식 유지

```
일부 코드의 eslint를 무시하는 방법

// 다음줄의 코드는 eslint 무시
eslint-disable-next-line

// 파일 전체를 무시
/*eslint-disable*/

```
